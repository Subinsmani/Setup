## Directory Structure

- **Ubuntu/**: Directory containing scripts for setting up various tools and configurations on Ubuntu.
- **setup.py**: Python script to automate the setup of your development environment.

## Getting Started

### Prerequisites

Before running the scripts, ensure you have the following installed on your system:

- Python 3.x
- Git

### Installation

Follow these steps to clone the repository and run the setup script:

1. Clone the repository:
   ```sh
   git clone https://github.com/Subinsmani/Setup.git
   cd Setup
   ```

2. Run the setup script:
   ```sh
   python3 setup.py
   ```
